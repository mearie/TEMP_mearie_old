메아리 플로는 메아리를 돌리는 웹 서버 어댑터입니다. 프레임워크랑은 다릅니다. 달라요.

MSIE는 (뭐 웬만한 경우 얘가 문제지만) 글꼴 fallback 알고리즘이 그야말로 거지같다. CSS 2.1의 15장 2절에서는 명시적으로 대응되는 글꼴을 하나의 문자 별로 처리할 것을 지정하고 있으나, MSIE는 처음으로 발견된 글꼴을 그대로 사용해 버린다. 운영체제들이 기본으로 지원하는 글꼴들로 충분히 approximate할 수 있는 경우에도 MSIE는 거지같이 반응한다.
